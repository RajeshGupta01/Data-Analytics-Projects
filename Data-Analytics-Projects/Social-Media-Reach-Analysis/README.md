# Social Media Reach Analysis

## Project Overview

This project analyzes social media engagement metrics such as likes, comments, shares, and reach to evaluate content performance.

## Features

* Engagement Analysis
* Reach Analysis
* Likes and Comments Tracking
* Best Performing Post Identification
* Data Visualization

## Technologies Used

* Python
* Pandas
* Matplotlib

## Files

* social_media.py
* social_media.csv
* README.md

## How to Run

1. Install Python.
2. Install required libraries:
   pip install pandas matplotlib
3. Run the program:
   python social_media.py

## Output

* Engagement Graph
* Reach Analysis Chart
* Performance Report
