# Sales Trend Visualization

## Project Overview

This project analyzes sales data and visualizes trends to help understand business performance and growth patterns.

## Features

* Monthly Sales Analysis
* Product-wise Sales Analysis
* Profit Analysis
* Sales Trend Visualization
* Business Insights

## Technologies Used

* Python
* Pandas
* Matplotlib

## Files

* sales.py
* sales.csv
* README.md

## How to Run

1. Install Python.
2. Install required libraries:
   pip install pandas matplotlib
3. Run the program:
   python sales.py

## Output

* Monthly Sales Graph
* Product Sales Chart
* Profit Analysis Report
