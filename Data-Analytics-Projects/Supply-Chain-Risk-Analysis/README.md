# Supply Chain Risk Analysis

## Project Overview

This project analyzes supply chain data to identify delivery delays, inventory shortages, and supplier risks. It helps improve supply chain efficiency.

## Features

* Supplier Performance Analysis
* Delay Detection
* Inventory Risk Analysis
* High-Risk Supplier Identification
* Data Visualization

## Technologies Used

* Python
* Pandas
* Matplotlib

## Files

* supply_chain.py
* supply_chain.csv
* README.md

## How to Run

1. Install Python.
2. Install required libraries:
   pip install pandas matplotlib
3. Run the program:
   python supply_chain.py

## Output

* Supplier Delay Graph
* Inventory Analysis
* Risk Assessment Report
