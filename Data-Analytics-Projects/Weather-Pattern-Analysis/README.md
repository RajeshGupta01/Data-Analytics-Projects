# Weather Pattern Analysis

## Project Overview

This project analyzes weather data to identify patterns in temperature, humidity, and rainfall. It helps users understand weather trends through graphical visualization.

## Features

* Temperature Analysis
* Humidity Analysis
* Rainfall Analysis
* Trend Visualization
* Statistical Summary

## Technologies Used

* Python
* Pandas
* Matplotlib

## Files

* weather.py
* weather.csv
* README.md

## How to Run

1. Install Python.
2. Install required libraries:
   pip install pandas matplotlib
3. Run the program:
   python weather.py

## Output

* Temperature Trend Graph
* Rainfall Analysis Graph
* Weather Statistics
